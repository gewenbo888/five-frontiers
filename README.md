# Five Frontiers · 五大前沿

A bilingual atlas of the five frontiers of human progress — Energy, Information, Spacetime, Matter, Biology — with the representative technologies under each, and the civilizational goal each one points toward.

## Links

- **Live:** [five-frontiers.psyverse.fun](https://five-frontiers.psyverse.fun)
- **Vercel:** auto-deployed on push to `main`
- **GitHub:** [github.com/gewenbo888/five-frontiers](https://github.com/gewenbo888/five-frontiers)

## Stack

Next.js 14 (App Router) · Tailwind CSS · static prerender. Fonts: Fraunces (display), Noto Serif SC (CJK), Inter (sans), JetBrains Mono (mono).

## Local

```bash
npm install
npm run dev      # http://localhost:3000
npm run build && npm start
```
